# Инструкция по заполнению операций для производства изделий из баннеров

## Введение

Этот документ описывает:
1. **Как заполнять операции в форме заказа** (фронтенд, CreateOrderForm)
2. **Как настраивать шаблоны операций в админке** (Admin Panel → Materials)

Операции настраиваются в **админке** для каждого материала отдельно. Ниже — полные инструкции по созданию и использованию операций для баннеров.

---

## Админка: Добавление операций к материалу

### Доступ к админке
1. Перейдите по маршруту `/admin` (требуются права администратора)
2. Выберите вкладку **"Materials"**
3. Кликните на материал (например, "Баннер 1000gr") чтобы открыть диалог
4. Перейдите на вкладку **"Операции"**
5. Нажмите **"Добавить операцию"**

### 1. Раскрой баннера ( Cutting )

#### Настройки операции:

**Основные поля:**
- **Название операции:** `Раскрой баннера` (или `Раскрой с подворотом`)
- **Описание:** `Раскрой баннера с припуском на подворот`
- **Тип операции:** `CUTTING` (Раскрой)
- **Базовая цена:** `1` (или ваша ставка за м²)
- **Ед. изм.:** `м²`
- **Коэффициент отходов:** `1.2` (или ваш коэффициент)
- **Требует размеры:** ✓ (checked)
- **Доп. материалы:** ✗ (unchecked)

**Параметры (добавить 4 параметра):**

| Поле | Значение |
|------|----------|
| `width` | **Ключ:** `width` • **Название:** `Ширина (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **Обязательное:** ✓ |
| `height` | **Ключ:** `height` • **Название:** `Высота (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **Обязательное:** ✓ |
| `marginWidth` | **Ключ:** `marginWidth` • **Название:** `Припуск ширина (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **По умолч.:** `50` • **Обязательное:** ✗ |
| `marginHeight` | **Ключ:** `marginHeight` • **Название:** `Припуск высота (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **По умолч.:** `50` • **Обязательное:** ✗ |
| `coefficient` | **Ключ:** `coefficient` • **Название:** `Коэффициент подвора` • **Тип:** `NUMBER` • **По умолч.:** `2` • **Обязательное:** ✗ |

**Формула расчета (в шаблоне продукта):**
```
(width + marginWidth*2) * (height + marginHeight*2) / 1_000_000 * coefficient
```

**Пример заполнения в заказе:**
- Ширина: 2000 мм
- Высота: 1000 мм
- Припуск ширина: 50 мм
- Припуск высота: 50 мм
- Коэффициент: 2
- **Результат:** `(2000+100) × (1000+100) / 1_000_000 × 2 = 4.84 м²`

### 3. Ламинирование/оклейка (Lamination)

#### Настройки операции:

**Основные поля:**

- **Описание:** `Ламинирование баннера`
- **Тип операции:** `LAMINATION`
- **Базовая цена:** `80` (ставка за м²)
- **Ед. изм.:** `м²`
- **Коэффициент отходов:** `1.05` (5% отходов)
- **Требует размеры:** ✓ (checked)
- **Доп. материалы:** ✗ (unchecked)

**Параметры (4 параметра):**

| Поле | Значение |
|------|----------|
| `width` | **Ключ:** `width` • **Название:** `Ширина печати (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **Обязательное:** ✓ |
| `height` | **Ключ:** `height` • **Название:** `Высота печати (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **Обязательное:** ✓ |
| `marginWidth` | **Ключ:** `marginWidth` • **Название:** `Припуск по ширине (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **По умолч.:** `20` • **Обязательное:** ✗ |
| `marginHeight` | **Ключ:** `marginHeight` • **Название:** `Припуск по высоте (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **По умолч.:** `20` • **Обязательное:** ✗ |

**Формула (с нахлестом):**
```
(width + marginWidth*2) * (height + marginHeight*2) / 1_000_000
```

**Пример заполнения в заказе:**
- Ширина печати: 2000 мм
- Высота печати: 1000 мм
- Припуск: 20 мм с каждой стороны
- **Результат:** `(2000+40) × (1000+40) / 1_000_000 = 2.1456 м²`

**Варианты:**
- Без нахлеста: оставьте припуски `0`
- Одностороннее ламинирование: коэффициент `1`
- Двустороннее: учтите в формуле или создайте отдельную операцию

### 4. Установка люверсов (Eyebolts) — с шагом

#### Настройки операции:

**Основные поля:**
- **Название операции:** `Установка люверсов` (или `Люверсы`)
- **Описание:** `Установка люверсов с шагом`
- **Тип операции:** `EYELETS`
- **Базовая цена:** `15` (цена за один люверс)
- **Ед. изм.:** `шт`
- **Коэффициент отходов:** `1` (не применяется)
- **Требует размеры:** ✓ (checked) — нужны для расчёта количества
- **Доп. материалы:** ✓ (checked) — можно добавить клей, скотч и т.д.

**Параметры (5 параметров):**

| Поле | Значение |
|------|----------|
| `width` | **Ключ:** `width` • **Название:** `Ширина изделия (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **Обязательное:** ✓ |
| `height` | **Ключ:** `height` • **Название:** `Высота изделия (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **Обязательное:** ✓ |
| `step` | **Ключ:** `step` • **Название:** `Шаг установки (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **По умолч.:** `500` • **Обязательное:** ✗ |
| `edgeDistance` | **Ключ:** `edgeDistance` • **Название:** `Отступ от края (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **По умолч.:** `50` • **Обязательное:** ✗ |
| `diameter` | **Ключ:** `diameter` • **Название:** `Диаметр люверса (мм)` • **Тип:** `NUMBER` • **Ед.изм.:** `мм` • **По умолч.:** `12` • **Обязательное:** ✗ |

**Формула расчёта количества (JEXL):**
```
helper.eyeletCount(width, height, step, edgeDistance, quantity)
```

**Алгоритм (backend):**
```java
// Конвертируем в мм
wmm = width * 1000
hmm = height * 1000

// Эффективные размеры с учётом отступа
effW = wmm - 2 * edgeDistance
effH = hmm - 2 * edgeDistance

// Количество точек по каждой стороне (округление вверх)
countW = ceil(effW / step)
countH = ceil(effH / step)

// Всего люверсов: 2 по ширине + 2 по высоте, минимум 4
totalPerBanner = max(4, 2*countW + 2*countH)
```

**Пример заполнения в заказе:**
- Ширина: 2000 мм
- Высота: 1000 мм
- Шаг: 500 мм
- Отступ от края: 50 мм
- Количество баннеров: 2
- **Результат:** `4 люверса на баннер × 2 шт. = 8 шт.`

**Примечание:** Если шаг не задан — используется 500 мм по умолчанию.

## Общие принципы заполнения в форме заказа (CreateOrderForm)

### Когда создаёте заказ:
1. Выбираете материал (например, баннер)
2. Вводите размеры **в миллиметрах** (Ширина × Высота)
3. Нажимаете **"Операция"** → выбираете нужную операцию
4. В форме операции заполняете параметры:
   - Поля с `Обязательное: ✓` — должны быть заполнены
   - Поля с `По умолч.` — можно оставить пустыми (подставятся значения по умолчанию)
5. Видите **предрасчёт** количества внизу формы
6. Нажимаете **"Добавить"** → операция добавляется к позиции
7. При необходимости добавляете несколько операций

### Калькуляция итога:
Материалы и операции считаются отдельно:
```
Итого по позиции = 
  (Материал: цена × кол-во × коэф.отходов) + 
  (Операция: цена × кол-во × коэф.отходов)
```

## Валидация полей

| Поле | Ограничения |
|------|-------------|
| `width`, `height` | > 0 |
| `marginWidth`, `marginHeight` | ≥ 0 |
| `coefficient` | 1 или 2 |
| `step` | > 0 (мин. 50) |
| `edgeDistance` | 30–50 мм |
| `diameter` | 8, 12 или 16 мм |
| `quantity` | > 0 |

---

## Важные замечания

1. **Единицы измерения:** все размеры вводятся в **миллиметрах**, но в БД хранятся в метрах (делим на 1000 или 1_000_000)
2. **Припуски:** учитывайте их в операции **раскроя**, не в печати
3. **Коэффициент отходов:** применяется к материалу (не к операции)
4. **Тип операции:** выбирайте из справочника (PRINT, CUTTING, LAMINATION, WELDING, EYELETS, INSTALLATION…)
5. **Формулы в JEXL:** можно использовать `width`, `height`, `helper.eyeletCount()`, арифметику `+−*/`, `ceil()`, `max()`, `min()`
